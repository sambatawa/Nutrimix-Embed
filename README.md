# Nutrimix - Website Pelet Ikan

Website e-commerce untuk Nutrimix, produsen pelet ikan premium dengan formula khusus untuk pertumbuhan optimal ikan budidaya.

## Start

```bash
npm install
npm run dev
http://localhost:3000
```

## Tech Stack

- **Frontend**: Next.js 16, React 19, TailwindCSS
- **Animasi**: Framer Motion
- **Auth**: Firebase, Google OAuth
- **Payment**: Midtrans
- **Icons**: Lucide React

## Struktur

```
src/
├── app/          # Next.js App Router
├── components/   # React components
└── lib/         # Firebase config
```

## Commands

```bash
npm run dev   
npm run build 
npm run start  
npm run lint  
```
